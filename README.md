# 💫 COSMOS 💫
- 자기 개발을 위해 쉽고 빠르게
팀원을 구할 수 있는 커뮤니티 플랫폼

## ✏ 프로젝트 목적 
- 기존 스터디 모집 사이트의 기능에 사용자가 다니고 있는
회사 및 경력 여부를 추가하여 같은 회사 사람들 또는 근무
하고 싶은 회사를 희망하는 사람들끼리 모여 스터디 및 
프로젝트를 진행하기 위한 팀원을 모집하는 서비스

## 💻 구현 기술 스택 
- JDK 11.0.15
- MySQL 8.0
- HTML, CSS, JS
- JQuery 3.6
- AJAX
- jsp
- OAuth 2.0 Google/Kakao
- Summernote api 
- Gmail api
- Apache Tomcat ver.9.0
 
## 💡 프로젝트에서 맡은 역할 
* DB 설계
* 퍼블리싱
  - 로그인
  - 회원가입
* 백엔드
  - 메인(배너, 게시글 목록) 구현
  - 게시글 작성, 수정, 삭제 구현
